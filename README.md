# CentOS7(さくらVPS)設定Ansible

さくらVPS(v4) + 通常インストールCentOS7を設定するAnsible。テスト用Vagrantつき。

## 設定内容（初期化）
- sshユーザー追加
- sshポート変更
- rootログイン禁止
- yum-cron追加（自動更新/通知あり）
